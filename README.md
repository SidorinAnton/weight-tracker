# Трекер веса

Сайт - [https://antonsvm.karpovdns.net](https://antonsvm.karpovdns.net)

## Функциональность:

- Добавляем метрики (вес, обхват) и отображаем их на графике и в таблице
- Устанавливаем цель (такой-то вес к такой-то дате)
